# Grid-Connected PV–Battery Microgrid with Smart-Inverter Support — Code and Data

Reproducibility archive (code and supplementary data) for the article:

> **Theoretical Formulation and Simulation-Based Verification of a Grid-Connected
> Photovoltaic–Battery Microgrid with Smart-Inverter Support for High-Irradiance
> Residential Applications in Saudi Arabia**
> A. Hakami, M. A. Khan, A. M. A. Alshehri, A. A. A. Asiri, A. K. Alhallafi.
> *Solar* (MDPI), manuscript **solar-4407776**.

This repository contains the independent **Python verification code**, the runnable
**MATLAB/Simulink model**, the **input resource data** for Riyadh, and the
**precomputed result files**. It reproduces every numerical result and design table
in the paper.

> **Not included:** the manuscript and the figure image files / figure-plotting
> scripts are intentionally excluded from this archive.

## Python — quick start
Requires Python 3.8+ with NumPy and SciPy.
```bash
pip install -r requirements.txt
cd code/python
python run_all.py          # writes the result JSON/NPZ files into code/python/
```
Individual stages can also be run on their own (e.g. `python microgrid_core.py`).
The core single-diode/EMS engine runs in a few seconds; the twelve-month analysis
(`month_analysis.py`) is the slowest stage (~1 min) because each month is iterated
to cyclic steady state.

## MATLAB / Simulink — quick start
Base **MATLAB R2024b** (or GNU Octave) for the `.m` scripts — **no toolboxes**;
**Simulink** only for `microgrid_lcl.slx`. See `code/matlab/README.md` for details.
```matlab
>> run_all                 % reproduces the study and prints a summary
>> build_microgrid_lcl_slx % (re)builds the Simulink model
>> run_thd_simulink        % runs it and reports the grid-current THD
```

## Input data
`data/riyadh_monthly.csv` holds the monthly-mean **global horizontal irradiance**
(kWh·m⁻²·day⁻¹) and **ambient temperature** (°C) for Riyadh (24.7° N, 46.7° E),
compiled from the K.A.CARE Renewable Resource Atlas and the Global Solar Atlas,
together with the resulting monthly performance. Annual GHI ≈ 2183 kWh·m⁻².

## Headline results reproduced
| Quantity | Value |
|---|---|
| Array power at STC | 6.24 kW (nameplate error 0.12 %) |
| Thermal derating 25 → 45 °C | 7.7 % (to 5.76 kW) |
| Annual self-sufficiency (Riyadh, optimised) | 51.8 % |
| PV self-consumption (annual) | 72.9 % |
| Grid-current THD, LCL vs L filter | 0.76 % vs 6.2 % |
| CAPEX | ≈ SAR 49,959 (≈ USD 13,322) |
| LCOE | ≈ SAR 0.52 /kWh |
| Simple payback (subsidised SEC tariff) | 23.2 yr |
